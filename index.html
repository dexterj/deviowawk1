Week 1

Having started Dev/Iowa Bootcamp, I'm more excited than before we began. Coming into this class I didn't know what to expect, I just knew I couldn't program. Now that I've gotten more hands on with javascript and some backend programming I can't wait to get started with the next eight weeks.

I've enjoyed spending time in js and feel like I can understand it more and more. I feel like I still need time with it, but I imagine that's going to be a big part of class. I like that we spent time in code academy on our own and had Thomas around to ask questions. I also enjoyed El Bandito.

I'm very intimidated by the back end programming. I'm glad we spent half a day on it so that I was able to get my head wrapped around how it's used and then had time to think about it. I look forward to learning other programming languages and doing more as we progress in the course.
